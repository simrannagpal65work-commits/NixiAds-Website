# Nixi Ads Website — Design Spec
**Date:** 2026-05-19  
**Stack:** React + Framer Motion (existing `test-website/` app)  
**Brand:** Nixi Ads — AI ad creative generator for D2C brands

---

## Overview

A modern, animated landing page for Nixi Ads built in React with Framer Motion scroll animations. Replaces the existing plain HTML site with a component-based, maintainable React app. Light & clean aesthetic with expressive Framer Motion animations.

---

## Design Decisions

| Decision | Choice |
|---|---|
| Visual style | Light & Clean — white background, orange (`#F97316`) accents, Inter font |
| Hero layout | Split 50/50 — text left, animated live generation UI right |
| Sections | Nav + 8 content sections + Footer (see below) |
| Animation style | Expressive & Dynamic — spring physics, staggered reveals, whileInView triggers |
| Framework | React (existing app) + Framer Motion (already installed) |

## Design System

### Colors
```css
--accent: #F97316;
--accent-gradient: linear-gradient(135deg, #F97316, #c2410c);
--text: #212121;
--text-2: #666;
--text-3: #aaa;
--bg: #fff;
--bg2: #fafafa;
--border: #ebebeb;
--accent-light: #fff7ed;
--accent-mid: #fed7aa;
--accent-dark: #9a3412;
--r: 18px;
--r-lg: 28px;
```

### Typography
- Font: Inter (Google Fonts, weights 300–900)
- H1: `clamp(48px, 6vw, 72px)`, weight 700, letter-spacing -2.5px
- H2: `clamp(28px, 4vw, 42px)`, weight 900, letter-spacing -1.5px
- Body: 15–16px, line-height 1.6–1.75

### Framer Motion Animation Patterns

**Scroll fade-up** (used on all section content):
```js
initial={{ opacity: 0, y: 40 }}
whileInView={{ opacity: 1, y: 0 }}
viewport={{ once: true }}
transition={{ duration: 0.6, ease: [0.22, 1, 0.36, 1] }}
```

**Stagger container** (used on grids/lists):
```js
variants={{
  hidden: {},
  show: { transition: { staggerChildren: 0.1 } }
}}
```

**Spring hover** (used on cards):
```js
whileHover={{ scale: 1.03, y: -4 }}
transition={{ type: 'spring', stiffness: 300, damping: 20 }}
```

**Hero entry** (on load, not scroll):
- Text side: `x: -40 → 0`, opacity 0→1, duration 0.7
- Widget side: `x: 40 → 0`, opacity 0→1, duration 0.7, delay 0.15

### Responsive Breakpoints
- `900px`: hero switches to single column, grids collapse to 1 col, nav links hide
- `700px`: reduced padding, font sizes clamp lower, badges hidden on mobile

## Sections

### 1. Nav
Fixed, frosted glass (`backdrop-filter: blur(20px)`, `background: rgba(255,255,255,0.9)`). Logo (Nixi-logo-trans.webp) left. Nav links center: How it works, Zero Prompts, Brand Sense, FAQ. "Join Waitlist" orange pill button right. Links hidden on mobile (`900px`).

### 2. Hero
Split 50/50 grid, `align-items: center`, `min-height: calc(100vh - 62px)`. Radial orange gradient background glow.

**Left:**
- Pill tag: "AI Ad Creative Engine" (orange)
- H1 with rotating word animation: "No `[prompt./brief.]` Just **ads.**" — word cycles with character-by-character slide-in/out animation (ported from current site)
- Subheading paragraph
- Primary CTA: "Join the Waitlist →"
- "Free to join · Early access coming soon" note

**Right:**
- Live generation widget — Mac-style window frame, runs a loop: type URL → brand scan checklist → spinning generator → ad image reveal → erase and repeat. Three brands cycle: velora, shoes, lumora (using existing asset images).
- 3 floating badges with `floatB` keyframe animation: ⚡ Under 2 minutes, 🎯 Meta-ready formats, 🎨 Brand-aware output

**Framer Motion:** Text side animates in from left on load, widget from right, with spring entry.

### 3. Ad Carousel
Full-width strip between Hero and How It Works. Label: "Ads generated by Nixi". Auto-scrolling track using `requestAnimationFrame`, seamless loop (duplicate set of images). Drag-to-scroll with momentum. Pause on hover. Images: carousel-ad1 through carousel-ad15 (excluding ad9, ad11, ad13). Framer Motion: whole section fades in on scroll.

### 4. How It Works
Section heading: "Three steps to a full campaign" / "No brief. No designer. No waiting."

3-column bento grid, each card has:
- Visual mockup at top (URL bar + scan tags / angle selector / stacked ad cards)
- Number badge (orange gradient)
- Title + description

Steps: 1. Drop in your product URL → 2. Pick your campaign angle → 3. Get campaign-ready ad creatives

Framer Motion: cards stagger in with spring physics.

### 5. Zero Prompts
Section heading: "You bring the product. Nixi brings the strategy."

Two-column comparison:
- **Left (grey):** "What other tools ask you" — 7 strikethrough items (target audience, tone, visual style, creative brief, selling points, colour palette, copy)
- **Right (orange tint):** "What Nixi asks you" — URL typing animation + generate button + "Nixi handles the rest" tag cloud (Brand voice, Visual DNA, Audience, Campaign angles, Headlines & copy, Ad format)

Framer Motion: left column slides in from left, right from right, on scroll.

### 6. Brand Sense
Background: `--bg2`. Section heading: "Every ad knows your brand."

Asymmetric 2-column grid:
- **Left (big card):** Orange-tinted border, "Not generic. Never off-brand." headline, description, `brandsense.webp` image
- **Right:** 2×2 grid of small feature cards: 🗣️ Brand Voice & Tone, 🎨 Visual DNA, 👥 Audience Profile, ⚔️ Competitor Awareness

Framer Motion: big card slides in from left, small cards stagger in from right.

### 7. Testimonials
Background: `--bg2`. Section heading: "Loved by D2C founders."

3-column grid of quote cards. Each card: star rating → blockquote → author (avatar initial + name + role). 3 testimonials with different coloured avatars (orange, indigo, green).

Framer Motion: cards stagger up on scroll enter.

### 8. FAQ
Background: `--bg2` with top border. Section heading: "Common questions."

Accordion — 7 items (same questions as current site). Click summary to expand/collapse. Framer Motion `AnimatePresence` + `motion.div` for smooth height animation. Orange `+` rotates 45° to `×` when open.

### 9. Final CTA
Full-width, gradient background (`#fff → #fff8f6`). Large centered heading: "Be first to experience *ads that actually convert*". Subtext. Big "Join the Waitlist →" orange button. "Free to join · Early access coming soon" note below.

Framer Motion: heading and button fade up with stagger on scroll.

### 10. Footer
Logo left, Privacy Policy + Terms of Use links center, "© 2026 Nixi. AI ad creative generator." right. Wraps on mobile.
